# Tech Stack

## Frontend

- Vue 3
- Vite
- Vuetify (required UI component library — do NOT use ElementPlus)
- JavaScript or TypeScript

## Backend

- Spring Boot 4.x
- Java 17+
- Spring Web
- Spring Data JPA

## Database

- MySQL or PostgreSQL

## DevOps

- GitHub Actions
- Docker
- Microsoft Azure (required deployment target)

## Testing

- Backend: JUnit, Mockito
- Frontend: Vitest / Cypress (optional)
